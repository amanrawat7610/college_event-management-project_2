# 🎓 College Event Management System

A complete web-based **College Event Management System** developed as a Final Year Major Project.  
This platform helps colleges manage events, student registrations, feedback, and event records digitally with an admin dashboard.

---

## 📌 Project Overview

The College Event Management System is designed to simplify the process of organizing college events.

The system provides:

- Admin authentication
- Event creation and management
- Student event registration
- Feedback collection
- Data export
- Responsive design for mobile and desktop
- AI chatbot assistance for users

The main goal of this project is to reduce manual paperwork and provide an efficient digital solution for college event management.

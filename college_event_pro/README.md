A secure and efficient web application built with **Flask** to manage student registrations and college events. This project was designed to streamline the coordination of campus fests and technical events.

## 🚀 Features
- **Dashboard:** At-a-glance view of total registrations and upcoming events.
- **Student Registration:** Seamless form for students to register for specific events.
- **Admin Control:** Secure signup/login to manage, add, or delete events and student records.
- **Security:** - Password hashing using `werkzeug`.
  - Protected routes with login decorators.
  - Master Key authentication for administrative access.
  - Environment variable integration for sensitive keys.

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Database:** SQLite (SQLAlchemy ORM)
- **Frontend:** HTML5, CSS3, Bootstrap
- **Tools:** Warp AI (Terminal), Git/GitHub
